# Multiple pathogen structure

Multiple pathogen structure

## Usage

``` r
multiple(data, case_timeseries, time = NULL, component_pathogen_timeseries)
```

## Arguments

- data:

  dataframe containing columns with all relevant data, or a time series
  object (ts, xts, zoo, tsibble, etc.)

- case_timeseries:

  Column name containing case counts. Must be numeric or a `units`
  object from the units package.

- time:

  name of column with time data. Required for non-time-series input
  data. Flexible format - can be date, index, or others, accepted as
  `index` identifiers in the `tsibble` time format. Optional when `data`
  is a time series class object (ts, mts, xts, zoo, zooreg, tsibble) as
  the time index will be automatically detected.

- component_pathogen_timeseries:

  vector of column names with additional pathogen case count timeseries
  to model. Must be numeric or a `units` object from the units package.

## Value

named list including pathogen_structure, pathogen_names, and data of
class `EpiStrainDynamics.pathogen_structure`

## See also

Other pathogen_structure:
[`single()`](https://acefa-hubs.github.io/EpiStrainDynamics/reference/single.md),
[`subtyped()`](https://acefa-hubs.github.io/EpiStrainDynamics/reference/subtyped.md)

  time = 'date',
  component_pathogen_timeseries = c('alpha', 'delta', 'omicron', 'other')
