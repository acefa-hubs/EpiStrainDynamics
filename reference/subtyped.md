# Subtyped pathogen structure

Subtyped pathogen structure

## Usage

``` r
subtyped(
  data,
  case_timeseries,
  time,
  influenzaA_unsubtyped_timeseries,
  influenzaA_subtyped_timeseries,
  other_pathogen_timeseries
)
```

## Arguments

- data:

  dataframe containing columns with all relevant data

- case_timeseries:

  name of column with total case data, this name will be used as the
  name of the pathogen

- time:

  name of column with time data. flexible format - can be date, index,
  or others, accepted as `index` identifiers in the `tsibble` time
  format.

- influenzaA_unsubtyped_timeseries:

  vector of column names with additional unsubtyped influenzaA case
  count timeseries

- influenzaA_subtyped_timeseries:

  vector of column names with additional subtyped influenzaA case count
  timeseries

- other_pathogen_timeseries:

  vector of column names with additional pathogen case count timeseries
  to model

## Value

named list including pathogen_structure, pathogen_names, and data of
class `EpiStrainDynamics.pathogen_structure`

## See also

Other pathogen_structure:
[`multiple()`](https://acefa-hubs.github.io/EpiStrainDynamics/reference/multiple.md),
[`single()`](https://acefa-hubs.github.io/EpiStrainDynamics/reference/single.md)
